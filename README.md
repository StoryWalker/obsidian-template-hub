# TemplateHub for Obsidian

TemplateHub is a plugin for Obsidian that enhances your template management workflow.

## Features

-   **Dedicated Template Folder**: Configure a specific folder for your templates, separate from Obsidian's core template plugin.
-   **Quick Insert Command**: Use the TemplateHub: Insert template command to quickly find and insert templates from your dedicated folder.
-   **Ribbon Icon**: Access the insert template command via a customizable icon in the sidebar.
-   **Configurable UI**:
    -   Show or hide the ribbon icon.
    -   Choose from any built-in Obsidian icon for the ribbon, with a mosaic of common icons for quick selection.

## Development

-   This plugin is built with TypeScript and esbuild.
-   Clone the repository.
-   Run 
pm install to set up dependencies.
-   Run 
pm run dev to start compilation in watch mode.
